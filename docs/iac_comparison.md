# IaC Tools Comparison

## Terraform
- Pros: Multi-cloud support, declarative, state management
- Cons: Learning curve, complex for simple tasks

## CloudFormation
- Pros: Native AWS integration, JSON/YAML
- Cons: AWS-only, verbose

## ARM Templates
- Pros: Native Azure integration
- Cons: Azure-only, JSON-based

## Conclusion
Terraform is best for multi-cloud scenarios due to its provider ecosystem.
